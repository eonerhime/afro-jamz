# Afro-Jamz

Afro-Jamz is a web platform that allows music producers to upload and sell beats to buyers. Each beat includes licensing terms that affect pricing and usage rights. Buyers can browse beats without an account but must log in to purchase and track their purchases.

## Features

- Producer account creation and profile management
- Upload beats with metadata and selectable licensing
- Browse and search beats as a guest
- Authenticated purchase of beats with license selection
- Commission tracking for platform and producer revenue

- Frontend: React.js + Tailwind CSS
- Backend: Node.js + Express/Fastify (lightweight)
- Database: SQLite (initially)

## Getting Started

1. Clone the repo
2. Install dependencies in frontend and backend
3. Run backend server
4. Run frontend dev server
